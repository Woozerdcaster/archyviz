# ArchyViz

Visualization tools for exploring relationships between burial practices and geographic distribution in Late Chalcolithic and Early Bronze Age Anatolia.

## Current Work

- Burial variable correlation analysis
- Geographic heatmaps
- Spatial visualization of mortuary practices
- Interactive archaeological data exploration

## Technologies

- Python
- Jupyter Notebooks
- Pandas
- Matplotlib
- Plotly
- GIS / spatial analysis
